Unified Teaching Station

The new Unified Teaching Station aims to stream line and simplify the current Teaching Station Senior, both in physical design by creating a smaller profile to allow for more teaching space, simplifying the number of moving parts, and updating the usable space on top. As well as the technology aspects by creating a step by step interface to help easier connect source materials/device to the overhead display. The Unified Teaching Station will still offer all the same functionality as the traditional Teaching Station such as intercom tech support and control panel as well as new features including a flat top design for easier placement of device and a barrier-free device shelf.
